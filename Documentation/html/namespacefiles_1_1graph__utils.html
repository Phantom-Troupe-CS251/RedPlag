Counts number of files present in each bin. 1/bin_size must be an integer. 0 < bin_size <= 1. Default value of bin_size is 0.10

Creates heat map of similarity values of files. X - axis and Y - axis represent the files. The colour of the block represents the similarity.

folder_path: Location where graph is to be stored
coloring: Coloring of heat map. Default is hot
img_format: Format in which image is to be stored. Default value is 'png'
